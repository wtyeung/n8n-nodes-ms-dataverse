#### [0.2.9](https://github.com/wtyeung/n8n-nodes-ms-dataverse/compare/0.2.8...0.2.9)

> 23 December 2025

**New Features:**
- **Custom Authentication**: Added support for custom environment URL and access token in Options
  - Allows using tokens from production environments in test/sandbox environments without OAuth2 setup
  - Works for both record operations and SQL queries via TDS
  - No OAuth2 credentials required when using custom authentication
- **Field Schema Viewer**: Added "View Table Fields (Reference Only)" dropdown
  - Browse all available fields with display names, logical names, types, and permissions
  - Shows Create/Update/Read permissions for each field
  - Helps discover correct field names for queries and mappings

**Improvements:**
- Table selector now shows logical names in parentheses (e.g., "Account (account)")
- Better error messages with URL details for debugging
- Clean trailing slashes from environment URLs automatically
- Fixed parameter reading for load operations with custom authentication

**Bug Fixes:**
- Fixed 404 errors when using custom authentication
- Fixed table and field loading with custom authentication
- Resolved circular dependency issues in field schema selector
