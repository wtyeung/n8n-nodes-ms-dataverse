#### [0.2.11](https://github.com/wtyeung/n8n-nodes-ms-dataverse/compare/0.2.10...0.2.11)

> 23 December 2025

**New Features:**
- Add image and file download support for GET operation
  - Auto-detect image fields (Virtual attributes with `_url` and `_timestamp`)
  - Auto-detect file fields (Virtual attributes with `_name`)
  - Download thumbnails (base64 from response) or full images (via PowerApps endpoint)
  - Download files via Web API `/$value` endpoint
  - Support manual field specification via comma-separated lists
  - Proper binary data formatting with descriptive property names
  - Handle empty/null fields gracefully

**Improvements:**
- Add metadata-based field detection using EntityDefinitions API
- Improve error handling with helpful messages for field dropdown failures
- Add workaround tips for authentication issues in field list
- Simplify Access Token description
- Add detailed error logging for debugging
- Clean up console debug statements

**Documentation:**
- Update README with comprehensive image and file download documentation
- Add examples for auto-detect and manual specification
- Document technical details of download endpoints
